### Divide And Conquer
- 문제를 더 작은 문제로 분할하는 과정(Divide)
- 각 문제에 대해 구한 답을 원래 문제에 대한 답으로 병합하는 과정(Merge)
- 더 이상 답을 문할하지 않고 곧장 풀 수 있는 매우 작은 문제(Base case)
